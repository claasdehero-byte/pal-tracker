# 🩺 Pflege-Kompetenztracker (PAL-Tracker)

Ein modernes, webbasiertes Tool für Auszubildende und Studierende in der Pflege, um ihre Kompetenzen und Praxiserfahrungen gemäß den LAGeSo-Vorgaben (Berlin) zu dokumentieren und zu tracken.

## 🚀 Features

- **Kompetenzkatalog**: Vollständige Liste der Pflegekompetenzen (A-M) zum einfachen Dokumentieren des Fortschritts.
- **Fortschrittsanzeige**: Visualisierung des Gesamtfortschritts durch ein interaktives Dashboard.
- **Tagesbeurteilung**: Integriertes System für tägliche Feedbacks von Praxisanleitern (PAL).
- **Arbeitsaufträge**: Verwaltung von individuellen Lern- und Arbeitsaufträgen.
- **Export-Funktionen**:
  - 📄 **PDF-Export**: Generierung eines professionellen Nachweises für die Schule oder den Arbeitgeber.
  - 📊 **CSV-Export**: Datensicherung für die weitere Verwendung in Tabellenkalkulationen.
- **Dark Mode**: Unterstützung für helles und dunkles Design.
- **Offline-fähig**: Alle Daten werden lokal im Browser gespeichert (`LocalStorage`).

## 🛠 Technologien

- **Frontend**: HTML5, CSS3, Vanilla JavaScript.
- **Typografie**: Inter (Google Fonts).
- **Libraries**: [html2pdf.js](https://github.com/eKoopmans/html2pdf.js) für den PDF-Export.

## 📥 Installation

Da es sich um eine reine Webanwendung handelt, ist keine Installation erforderlich:

1. Repository klonen oder als ZIP herunterladen.
2. `index.html` in einem modernen Browser deiner Wahl öffnen.

## 📖 Nutzung

1. **Persönliche Daten**: Trage deinen Namen, Kurs und Einsatzzeitraum ein.
2. **Kompetenzen**: Hake erledigte Aufgaben ab (Gesehen, Unter Aufsicht, Selbstständig).
3. **Beurteilung**: Lass deinen PAL Tagesfeedbacks direkt im Tool eintragen.
4. **Export**: Nutze am Ende deines Einsatzes den PDF-Export, um deine Nachweise zu sichern.
